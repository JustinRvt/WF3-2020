Les transitions

Il existe 2 façons d'animer en CSS, on utilise les transitions ou les keyframes.

Les transitions sont plus faciles à manier, elles nous permettent de passer d'un état à un autre. Pour cela, nous avons besoin d'informations :

  • Une valeur initiale
  • Une valeur finale
  • Une durée
  • Un événement déclencheur
  • Une ou plusieurs propriétés CSS
