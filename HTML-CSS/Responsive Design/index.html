Responsive Design

Les règles disponibles :

  • media :
    - screen (l'écran de base)
    - print (impression)
    - all (tous les types d'écrans)
    - Obsolètes depuis Media queries lvl 4:
    - projection (projecteur)
    - tv (télévision)
  • width : largeur de la fenêtre
  • height: hauteur de la fenêtre
  • orientation : paysage ou portrait
  • device-width : largeur de l'appareil
  • device-height : hauteur de l'appareil

On peut rajouter des min- ou des max- devant ces règles.

min-width signifie largeur minimale tandis que max-width signifie largeur maximale.


On peut combiner toutes ces règles avec :

  • and (et)
  • not (non)
  • only (seulement)
