Les fonctions

Comprendre les fonctions

Une fonction est un bout de code que l'on nomme, que l'on appelle et qui va exécuter le code qu'elle contient.

Beaucoup de fonctions qui vont avoir besoin de variables pour fonctionner, lorsque l'on crée ou que l'on appelle une fonction, on indique les variables dont elle a besoin : c'est définir les paramètres.

On attribue des valeurs pour ces paramètres que l'on nomme les arguments d'appel.

Enfin, nous fonction nous donne un résultat, sa valeur de retour. Si nous avons une fonction qui calcule le nombre de mots dans un string :

  • le paramètre va être le string où l'on compte les mots.
  • l'argument sera tout string attribué à la fonction quand on l'appelle.
  • la valeur de retour sera le nombre de mots.

Méthodes d'instance et propriétés

Nous avons utilisé les classes pour créer des objets avec certaines propriétés, maintenant, nous allons ajouter des méthodes d'instance à ces classes pour augmenter leur potentiel.

Une méthode d'instance est une fonction qui fait partie du classe et qui agit sur une instance de cette classe.

Les méthodes dites statiques

Il s'agit d'un type spécial de méthode qui peut ajoutée à une classe. Elle est différente dans le sens où elle ne sera pas liée à une instance à la classe elle-même. C'est un peu une sorte de helpers. On s'en sert de boite à outils pour des fonctions que l'on utilise souvent.
